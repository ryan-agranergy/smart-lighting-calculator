# 智能照明计算器 (Smart Lighting Calculator)

## 项目简介

这是一个智能照明计算器应用，用于帮助用户计算和优化照明系统的能源消耗。

## 功能特点

- 照明能耗计算
- 照明系统优化建议
- 节能方案比较
- 投资回报期分析

## 安装说明

1. 克隆项目到本地：

```bash
git clone https://github.com/ryan-agranergy/smart-lighting-calculator.git
cd smart-lighting-calculator
```

1. 安装依赖：

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行应用：

```bash
python app.py
```

1. 在浏览器中访问：

```text
http://localhost:5000
```

## 技术栈

- Python
- Flask
- SQLite
- HTML/CSS/JavaScript
